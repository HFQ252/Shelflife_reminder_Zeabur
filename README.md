# 商品保质期临期提醒系统

一个基于Web的商品保质期管理系统，用于跟踪和管理商品的保质期，提供临期提醒功能。

## 功能特性

- 📦 商品信息管理（SKU、名称、保质期、库位）
- 📅 生产日期和到期日期跟踪
- ⚠️ 临期商品自动提醒
- 🔍 快速SKU查询和自动补全
- 📊 多维度数据展示（临期商品、所有商品、商品数据库）
- 📱 响应式设计，支持移动端
- 🔄 实时数据刷新
- 📋 重复记录检测

## 技术栈

- 前端：HTML5, CSS3, JavaScript, Bootstrap 5, Bootstrap Icons
- 后端：Node.js, Express.js
- 数据库：SQLite3
- 部署：Zeabur
